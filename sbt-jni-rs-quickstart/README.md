# call-rs

Example calling library written by Rust.

## Environment
- sammyne/scala:2.12.15-jdk8-rs1.60-ubuntu20.04

## Quickstart

```bash
sbt core/run
```

Ok output sample goes as

```bash
Hello, World!
457
579
```
